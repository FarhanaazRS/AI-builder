# 🚀 Quick Start

## 🔑 Prerequisites

Make sure the following are installed:

- Python 3.9+
- Node.js 16+
- PostgreSQL
- Git

---

# 1️⃣ Clone Repository

```bash
git clone https://github.com/FarhanaazRS/AI-builder.git

cd AI-builder
```

---

# 2️⃣ Backend Setup

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```

Backend runs at:

```text
http://localhost:8000
```

---

# 3️⃣ Frontend Setup

```bash
cd frontend

npm install

npm start
```

Frontend runs at:

```text
http://localhost:3000
```

# 💡 How It Works

## 1. Design Phase
- Drag components from the palette onto the canvas
- Connect components to define workflow execution
- Configure component settings and parameters

---

## 2. Validation & Testing
- Click **Build Stack** to validate workflow logic
- Use **Test Workflow** to execute sample queries
- Debug and refine workflow connections

---

## 3. Deploy & Use
- Launch AI assistants using **Chat with Stack**
- Query uploaded documents
- Interact with intelligent AI workflows
- Monitor workflow execution

---

# 🧠 Core Workflow Architecture

```text
User Query
    ↓
Workflow Builder (React Flow)
    ↓
Knowledge Base / Web Search
    ↓
Embedding + Retrieval Layer
    ↓
LLM Engine (GPT / Gemini)
    ↓
Output Chat Interface
```
